# 🏏 CricScorePro - End-to-End Cricket Management System

CricScorePro is a modern web application designed to manage cricket matches and track live scores in real-time. It features a powerful Spring Boot backend and a highly responsive React frontend.

## 🚀 Key Features
- **Real-Time Live Scoring:** Track matches with ball-by-ball updates.
- **Dynamic Dashboard:** Categorized views for Live, Upcoming, and Completed matches.
- **Admin Management:** Dedicated panel to manage Teams, Players, and Tournaments.
- **Advanced Analytics:** In-depth analysis of player performance and match statistics.
- **CI/CD Integration:** Automated testing and Docker image creation via GitHub Actions.

---

## 🛠️ Tech Stack
- **Frontend:** React.js, Tailwind CSS
- **Backend:** Java 17, Spring Boot, Spring Data JPA, Hibernate
- **Security:** Spring Security & JWT Authentication
- **Database:** MySQL / H2 (for testing)
- **CI/CD:** GitHub Actions & Docker
